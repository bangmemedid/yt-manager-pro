Kebijakan Privasi

Terakhir diperbarui: 05 Feb 2026

Aplikasi bangmemed.id - YouTube Manager Pro digunakan untuk membantu pengguna melihat ringkasan statistik channel YouTube dan mengelola data terkait melalui YouTube API dan Google OAuth.

Data yang Dikumpulkan

  • Informasi akun Google dasar (mis. email/identitas) melalui Google OAuth.
  • Data YouTube yang diizinkan oleh pengguna (mis. statistik channel, daftar channel, metrik penayangan).

Penggunaan Data

  • Menampilkan dashboard statistik dan daftar channel yang terhubung.
  • Mengelola sesi login (token OAuth) agar aplikasi dapat mengakses data sesuai izin pengguna.

Penyimpanan & Keamanan

Token OAuth digunakan untuk mengakses API atas nama pengguna. Kami berupaya menjaga keamanan token dan data dengan praktik keamanan yang wajar. Jangan membagikan akses akun Anda kepada pihak lain.

Berbagi Data

Kami tidak menjual data pengguna. Data hanya digunakan untuk fungsi aplikasi dan tidak dibagikan ke pihak ketiga, kecuali diwajibkan oleh hukum.

Penghapusan Akses

Anda dapat mencabut akses aplikasi kapan saja melalui pengaturan Google Account: myaccount.google.com/permissions.
