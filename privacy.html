Privacy Policy

Terakhir diperbarui: 7 Februari 2026

1. Informasi yang Kami Kumpulkan

YouTube Manager Pro menggunakan Google OAuth API untuk mengakses data akun YouTube Anda. Kami hanya membaca data statistik (Subscriber, Views, Videos) dan data analitik (Realtime views).

2. Penyimpanan Data

PENTING: Kami tidak memiliki server database. Semua data akses (Access Tokens) disimpan secara lokal di browser Anda (Local Storage). Kami tidak pernah mengirimkan data Anda ke server pihak ketiga mana pun.

3. Penggunaan Data

Data yang ditarik hanya digunakan untuk ditampilkan pada dashboard Anda guna memudahkan manajemen multi-channel secara real-time.

4. Keamanan

Keamanan data bergantung pada perangkat Anda. Harap pastikan Anda menggunakan perangkat pribadi dan melakukan "Logout Lokal" jika ingin menghapus seluruh data akses dari browser.
